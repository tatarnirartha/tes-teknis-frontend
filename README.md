## 1. Install some npm packages
```
npm install
```

### 2. Start php local server
```
php artisan serve
```

### 3. Compiles vue.js
```
npm run watch
```

### 4. Start JSON Server
```
json-server --watch ./Mocking/JSON/db.json 
```
